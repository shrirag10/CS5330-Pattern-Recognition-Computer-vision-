# Setup Instructions

This directory is not checked into git (large binaries).

Download the prebuilt CPU package from:
  https://github.com/microsoft/onnxruntime/releases/tag/v1.26.0

File to download: onnxruntime-linux-x64-1.26.0.tgz

Extract it at Projects/project1/ so the path becomes:
  Projects/project1/onnxruntime-linux-x64-1.26.0/include/
  Projects/project1/onnxruntime-linux-x64-1.26.0/lib/
